1. Alibaba Open-Sources Qwen-Image-Layered: AI Image Generation Moves from Pixel Prediction to Structural Recomposition

On July 23, Alibaba's Qwen team officially open-sourced Qwen-Image-Layered, a groundbreaking image generation model that achieves Photoshop-level layer understanding and generation for the first time in an AI model. This breaks the "flat thinking" paradigm of mainstream vision models.

Qwen-Image-Layered features a proprietary RGBA-VAE encoder developed by the Qwen team, which introduces an "Alpha channel" representing transparency layers into traditional RGB images, giving the model a true concept of layers. Combined with the innovative VLD-MMDiT architecture and unique "layer-level 3D position encoding," the AI can automatically infer background textures behind occluded regions, achieving deep understanding of layer structure and spatial relationships.

The core problem with traditional AI image generation has been that modifying any element required complete regeneration — the classic "tug one hair and the whole body moves" problem. Qwen-Image-Layered fundamentally solves this, enabling nearly "zero-drift" precise AI image editing. Designers can now move, scale, or regenerate specific layer elements while keeping backgrounds or subjects completely unchanged.

To train this capability, the Qwen team extracted real layering logic from millions of professional Photoshop (PSD) files, instilling the AI with a professional designer's "layered thinking" from the start. Qwen-Image-Layered is now available open source on ModelScope and HuggingFace for free commercial use.

To date, Alibaba has open-sourced nearly 400 Qwen models with over 700 million global downloads and more than 180,000 derivative models, making it the world's top open-source model provider. The Qwen model family now serves over 1 million enterprise customers.

2. OpenAI Signs $38 Billion AWS Computing Deal, Ending Microsoft Cloud Exclusivity

OpenAI announced July 23 that it has signed a $38 billion computing procurement agreement with Amazon Web Services (AWS), marking the first strategic partnership between the $500 billion AI startup and the world's largest cloud infrastructure provider. The deal formally ends Microsoft's exclusive cloud relationship with OpenAI.

Under the agreement, OpenAI will immediately begin workloads on the AWS cloud platform, initially deploying hundreds of thousands of Nvidia GPUs across U.S. data centers. Dave Brown, AWS Vice President of Compute and Machine Learning Services, stated: "We are deploying entirely independent computing capacity. Part of it is already available, and OpenAI is using it."

OpenAI had maintained an exclusive cloud relationship with Microsoft since 2019, with Microsoft investing a cumulative $13 billion. In January, Microsoft announced it would no longer serve as OpenAI's exclusive cloud provider, shifting to a "right of first refusal" model. Last week, following renegotiated commercial terms, Microsoft's priority status officially ended.

OpenAI has been on an infrastructure investment spree, announcing partnerships with Nvidia, Broadcom, Oracle, and Google totaling approximately $1.4 trillion. Notably, despite expanding its cloud partners, OpenAI confirmed it will continue purchasing Microsoft Azure services at scale — last week reconfirming an additional $250 billion in Azure cloud resources.

For Amazon, the deal is strategically significant — AWS already has a close partnership with OpenAI competitor Anthropic, having invested billions and building an $11 billion data center campus in Indiana dedicated to Anthropic workloads. Securing both OpenAI and Anthropic further solidifies AWS's dominant position in the AI cloud market.

3. BAAI Releases RoboBrain Orca World Model, AI Achieves World State Evolution Prediction for the First Time

Beijing Academy of Artificial Intelligence (BAAI) released on July 23 its new multimodal world model RoboBrain Orca (Wujie·RoboBrain Orca), breaking through the technical path of single-modality output prediction (text, images, or actions) to achieve holistic world state evolution prediction for the first time, opening a new technical route for AGI foundation model development.

The core concept behind RoboBrain Orca is that if AI can simulate the operational rules of the world internally, it transforms from a passive response tool into an active reasoning, planning, and predicting agent. The model integrates vision, language, images, text instructions, and task intent into a unified world latent representation space, enabling it to both predict future evolution paths from current scenarios and simulate multiple possibilities under different conditions.

The research team designed a "conscious + unconscious" dual-path complementary training system:

  • Unconscious learning: Leveraging massive real-world video footage to autonomously perceive object dropping, movement, spatial interaction, and other physical world rules without annotation
  • Conscious learning: Using text events and visual QA annotated data to learn semantically clear state transition logic and event temporal relationships

RoboBrain Orca was trained on 125,000 hours of real video footage and 160 million event annotations. The model can adapt to multiple decoder types: text decoding excels at dynamic changes and state transition logic reasoning; image decoding can accurately restore real scene interaction predictions; robot action decoding can significantly improve downstream robot complex task generalization without requiring pre-learned action labels.

In the first half of 2026, the global "world model" track saw investment explode, with over 40 billion yuan flowing in, including 46 financing rounds in China involving 33 companies racing ahead. Industry observers note that when AI truly begins simulating world changes in its "mind," the distance to AGI will no longer be measured in years.
