科技新闻OpenAI微软硬件AI代理腾讯云

⚡ 7月16日科技综述：OpenAI发布首款硬件Codex Micro发光键盘、微软Build 2026推"梦中神机"、腾讯云DeepSeek-V4最高降价97.5%

一、OpenAI意外跨界硬件：首款品牌产品竟是发光键盘

OpenAI在2026年7月悄然发布其首款品牌硬件——并非传闻中的AR眼镜或AI手机，而是一把名为Codex Micro的发光键盘。这款产品定位为AI代理（Agent）时代的"指挥中心"，旨在通过视觉化的键帽灯光实时反映多个AI代理的运行状态。

设计理念：让键盘成为AI仪表盘

Codex Micro的设计源于一个核心痛点：现代开发者往往需要同时运行多个AI代理——代码审查、文档处理、日志监控等在后台并行运转，切换查看状态极为繁琐。OpenAI的解决方案是让键盘本身成为信息显示器：每个键帽下方集成微型RGB LED，通过不同颜色和闪烁模式实时反映代理活动状态。代码审查代理对应蓝色呼吸光，遇到报错则变为红色急促闪烁，用户无需抬头即可掌握全局。

技术细节与定价争议

根据OpenAI官方博客，Codex Micro采用自研AgentMonitor固件与PC上的OpenAI Agent SDK通信。键盘内置低功耗ARM处理器和Wi-Fi模块，可直接接收云端代理状态码。空格键左侧新增触控条用于快速切换代理焦点，右侧旋钮调节全局工作负载阈值。

然而定价引发巨大争议——基础版售价349美元，Pro版高达599美元，超过许多高端机械键盘。OpenAI产品主管Mira Murati解释称："Codex Micro不是普通键盘，它是物理形态的AI状态仪表盘。"但业界分析师指出，该功能完全可通过软件插件或第二显示屏实现。

值得注意的是，这并非OpenAI完全自研。据Ars Technica调查，Codex Micro系与著名外设品牌Keychron合作研发，Keychron负责底盘和机械轴体，OpenAI提供电子模块和固件。TechCrunch称这是一次"限量合作"，更多是OpenAI进军硬件的序幕而非主力产品。

二、微软Build 2026：七款自研模型 + "梦中神机"Dev Box

微软在刚刚举办的Build 2026开发者大会上集中发布了自研模型家族、本地AI开发设备和新型AI终端形态，全面宣示其在AI基础设施层的布局。

MAI模型家族一次性亮相七款

微软一次性推出了七款MAI（Microsoft AI）自研模型，覆盖推理、代码、图像、语音和转录五大方向：

  • MAI Thinking 1：推理模型，采用稀疏MoE架构，活跃参数35B，总参数量约1T，支持256K token上下文
  • MAI Code 1 Flash：端到端训练代码模型，在SWE Bench Pro上得分51.2%，高于Claude Haiku 4.5的35.2%，已向GitHub Copilot个人用户推出
  • MAI Image 2.5：图像模型，已进入PowerPoint，计划扩展至OneDrive
  • MAI Transcribe 1.5：转录模型，支持43种语言
  • MAI Voice 2：语音模型，支持15种语言，可通过短样本适配声音

Surface RTX Spark Dev Box：纳德拉口中的"梦中神机"

硬件方面，微软推出面向开发者的Surface RTX Spark Dev Box，CEO萨提亚·纳德拉将其称为"dream machine（梦中神机）"。该设备基于英伟达RTX Spark平台，提供1 petaflop AI算力、20个CPU核心和128GB统一内存，热设计功耗仅100W，计划今年秋季推出。英伟达CEO黄仁勳通过视频连线表示："PC正在从个人电脑向个人AI演进。"

微软同时展示了代号Project Solara的两类参考设备：基于联发科芯片的桌面工作终端（支持身份识别、语音交互和Microsoft 365 Copilot接入），以及采用高通可穿戴芯片的数字工牌（面向医疗、现场采集等移动工作场景）。

三、OpenAI扩展Codex生态：插件覆盖投行、数据分析等六大方向

在昨日的"Intelligence at Work"直播活动上，OpenAI宣布将Codex核心能力直接整合进ChatGPT应用，同时推出角色专属插件、Sites共享网站功能及标注功能，将使用场景从软件开发延伸至知识工作领域。

官方数据显示，Codex每周活跃用户已超500万，其中非开发者用户约占20%，增速超过开发者用户的3倍。

六大插件方向

OpenAI上线6款角色专属插件，集成62款应用与110项技能，覆盖以下方向：

  • 数据分析
  • 创意生产
  • 销售
  • 产品设计
  • 公开市场股权投资
  • 投资银行

后续还将推出企业财务、私募股权、法律等更多方向，并计划开放合作伙伴生态。Sites功能面向企业版和商业版用户开放预览，支持将分析结果和方案规划生成可分享的交互式网页。

四、腾讯云DeepSeek-V4系列最高降价97.5%

腾讯云发布公告，旗下智能体开发平台即日起对DeepSeek-V4系列模型价格进行大幅下调，最高降幅达97.5%，标志着国内AI大模型价格战再度升级。

  • DeepSeek-V4-Pro：推理输入与输出价格均降75%，分别从每千tokens 0.012元、0.024元降至0.003元、0.006元
  • 缓存命中价格：从每千tokens 0.001元降至0.000025元，降幅高达97.5%
  • DeepSeek-V4-Flash：缓存命中价格从每千tokens 0.0002元降至0.00002元，降幅90%

这是继阿里云、百度智能云之后，又一家国内云厂商大幅下调AI模型推理价格。分析认为，随着开源模型能力逼近闭源水平，云厂商正通过价格战抢占推理市场份额，开发者成为最大受益者。

五、Vint Cerf离开Google后投身AI代理身份标准

互联网奠基人之一、"TCP/IP之父"Vint Cerf在离开供职20年的Google后，宣布担任Innovation Labs顾问，推动AI代理开放身份认证标准DNSid。该项目旨在为互联网上的AI智能体建立统一身份识别和审计框架。

Cerf称，AI代理的识别问题至关重要："它们拥有什么权限？谁对它们的行为负责？"Innovation Labs提出的DNSid系统，利用加密证明将每个AI代理与现有互联网域名绑定，并记录其注册时间线。该公司表示正在与多家超大规模云厂商和身份公司进行试点。

"我意识到在这个命名和识别日益重要的时期，我或许能帮上忙。这主要是由AI代理引发的——它们的权限来源、责任归属以及信任建立方式。"——Vint Cerf
