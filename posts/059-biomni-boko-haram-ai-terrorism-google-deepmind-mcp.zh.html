⚡ 7月12日AI快讯：斯坦福Biomni登Science改写生物医学研究、博科圣地使用AI策划袭击、Google DeepMind为Agent添加MCP

⚡ 斯坦福大学通用生物医学AI Agent Biomni登上Science期刊，35分钟完成人类专家3周数据分析，已在全球1万+实验室投入实战；CASP首次实地证据报告揭露博科圣地系统性使用ChatGPT、Claude、Gemini等前沿AI规划袭击、设计爆炸装置，伊斯兰国甚至派遣AI培训师；Google DeepMind为Gemini API Managed Agents添加强力新功能：后台执行、远程MCP服务器连接等四项升级，MCP协议已跨越1.3万服务器大关。

🧬 斯坦福Biomni登Science：AI生物医学「联合科学家」改写研发范式

斯坦福大学计算机科学教授Jure Leskovec团队在Science期刊发表论文，正式推出通用生物医学AI Agent——Biomni。这绝非普通聊天机器人：Biomni整合了150个专业生物医学工具、105个软件包和59个数据库，覆盖bioRxiv定义的全部25个生物医学子领域，从遗传学到神经科学无所不包。用户只需用自然语言描述研究问题——例如「为什么这些患者对药物反应不同？」——Biomni便会自主检索文献、形成假设、选择数据集、编写分析代码、解释结果并建议后续实验。在一项真实测试中，Biomni仅用35-40分钟就完成了对458份穿戴设备数据文件（连续血糖监测、饮食记录、活动量数据）的清洗、统一、可视化和模式发现，而人类专家估计需要3周（约60+小时）。论文第一作者兼项目架构师Kexin Huang（斯坦福计算机科学博士，现创立Phylo公司推进商业化）表示，Biomni在400多项真实研究任务中达到「专家级准确度」。更具意义的是，已有1万+学术和工业实验室在使用原型系统，使其成为生物医学领域部署最广的AI联合科学家。Jure Leskovec强调：「生物医学的瓶颈不是智力或想法，而是机械性体力劳动。Biomni能在几分钟内完成这些工作。」

⚠️ CASP报告首次实地证实：博科圣地系统性使用前沿AI策划袭击

安全与防护中心（CASP）发布迄今为止最详尽的恐怖组织AI使用实地调查报告——《"上帝帮助了我们，AI也会"：恐怖组织博科圣地如何使用前沿AI》。基于2025-2026年在尼日利亚东北部对27名前博科圣地成员进行的57次半结构化深度访谈，报告首次提供了一手证据：博科圣地的两个分支——伊斯兰国西非省（ISWAP）和JAS——均系统性使用ChatGPT、Claude、Gemini、Grok、Meta AI和DeepSeek等前沿AI，用于规划袭击、设计爆炸装置、武器故障排除和改善行动安全。IS甚至派遣7名教官赴ISWAP进行包括AI在内的专项培训。一名前ISWAP指挥官直言：「你打字提问或用语音问——比如'我怎么做炸弹？'——它就详细告诉你。就像一个人形机器人！我们大量使用。」报告显示，在ChatGPT发布后约两年内，两个派别已从初次接触到设立专门AI部门和融入作战常规流程。安全护栏并未阻止滥用：尽管存在限制，但受访者形容限制「可以管理」而非「禁止」。一名指挥官表示，受过培训的成员「知道如何绕过障碍」。283起AI滥用事件已被记录在案。报告警告：恐怖组织的AI采用速度、系统性和广度远超此前分析认知。

🔧 Google DeepMind大幅升级Agent平台：后台执行和MCP成标配

Google DeepMind为Gemini API Managed Agents新增四项关键能力。最大亮点是后台执行（Background Execution）：Agent现可异步运行无需保持HTTP连接，彻底摆脱了开发者必须维持长连接处理文档分析、多步数据管线等长时间任务的痛点。其次是远程MCP服务器连接：Agent可直接连接内部数据库或第三方API——MCP协议于2026年6月已跨越1.3万+服务器大关，7月8日Anthropic、OpenAI、Block以Linux Foundation为依托正式背书MCP，同日Google Cloud和AWS（7月7日）分别推出自家MCP服务器。第三项是自定义函数与沙箱工具并行使用。第四项更是生产环境的痛点解药：凭证在交互间可刷新，不会丢失沙箱状态。这四合一升级直接对标Anthropic Claude Code的MCP生态，Google选择拥抱对手协议而非自推标准，信号明确：MCP已成为Agent平台的入场券。分析指后台执行功能对标AWS Step Functions的成功模式，而凭证刷新是「沉睡的杀手级功能」——企业部署中Agent状态管理一直是头号运维难题。

📊 今日小结

7月12日的AI头条交织出令人深省的图景：斯坦福Biomni展示了AI赋能科学发现的巨大潜力——1万+实验室已在用它加速生物医学研究，35分钟vs 3周的生产力跃迁令人振奋；而CASP报告则揭示了AI技术被恶意使用的另一面——恐怖组织在两三年内便完成了从初次接触到制度化使用的快速演进，安全护栏远非不可逾越。与此同时，Google DeepMind选择拥抱Anthropic的MCP协议而非自建标准，说明AI行业的互联互通正在超越商业竞争。这些故事共同指向一个核心命题：在AI能力快速提升的时代，如何确保这项技术被导向建设性而非破坏性的方向，是全行业需要共同面对的根本挑战。
