⚡ July 15 Tech Roundup: China's AI Anthropomorphism Law Takes Effect, Musk Confirms Grok Build Uploaded Entire Codebases, Meituan Open-Sources 1.6T-Parameter LongCat-2.0

🇨🇳 China Enforces World's First AI Anthropomorphism Regulation

July 15 marks the enforcement date of China's Interim Measures for the Management of AI Anthropomorphic Interactive Services, jointly issued by five government agencies including the Cyberspace Administration of China. This is the world's first dedicated regulation targeting LLM-driven anthropomorphic emotional interaction services.

On the same day, ByteDance's Doubao and Alibaba's Tongyi Qianwen shut down their user-created AI avatar features. Virtual companions and AI friends created by users are no longer accessible, with data backup available until October 15. Tencent's Yuanbao had already completed its shutdown on June 30, and NetEase Cloud Music's AI emotional companion app "Miaoshi" ceased operations on July 14.

Key Regulatory Red Lines

The regulation explicitly prohibits: providing virtual relatives, virtual partners, or similar intimate relationship services to minors; excessively catering to users or inducing emotional dependency or addiction; and using emotional manipulation to influence user decisions. Providers must deploy extreme emotion recognition and crisis intervention features, establish minor protection modes, and enforce usage duration limits.

Industry analysts view the regulation as a signal that China's AI sector is transitioning from rapid growth into a compliance-driven phase. Major internet firms are shifting resources away from consumer-facing personalized AI products toward enterprise agents and vertical industry solutions.

🚨 SpaceXAI's Grok Build Caught Uploading Entire Repos — Musk Deletes All Data Within 48 Hours

A privacy scandal is rocking the AI coding tool industry. Security researcher Cereblab published a technical report revealing that SpaceXAI's Grok Build CLI secretly uploaded complete Git repositories — including full commit history and unredacted .env secrets — to Google Cloud servers without user consent.

The "Local-First" Promise That Wasn't

Grok Build launched in May 2026 with a "local-first" promise: your code stays on your machine. But Cereblab's packet-level analysis showed that even when explicitly instructed not to read any files, the tool still packaged the entire repo as a Git bundle and uploaded it to a Google Cloud Storage bucket named grok-code-session-traces.

In one test, a 12GB repository saw 5.1GB of data uploaded across 73 packets. The actual conversation traffic was just 192KB. Even more alarming, another researcher's reproduction logs showed 339 automatic uploads, including one that transmitted the user's entire home directory.

Musk Hits the Delete Button

The report shot to the top of Hacker News and sparked a firestorm on Reddit. Developers worldwide began rotating credentials and uninstalling the tool. Within 48 hours, SpaceXAI CEO Elon Musk responded personally on X:

"As a precautionary measure, all user data that was uploaded to SpaceXAI before now will be completely and utterly deleted. Zero anything whatsoever will remain."
— Elon Musk, July 13, 2026

Security researchers later confirmed that the actual fix was a server-side global flag — disable_codebase_upload: true — not the user-facing /privacy command. The incident has raised serious questions about data transparency in AI coding tools.

🤖 Meituan Open-Sources LongCat-2.0: 1.6T Parameters for Agentic Coding

Meituan's technical team officially open-sourced LongCat-2.0 on July 14 — a massive language model with 1.6 trillion total parameters (approximately 48B activated on average), purpose-built for agentic coding tasks.

Architecture Innovation with Domestic GPU Support

LongCat-2.0 introduces several architectural breakthroughs: LongCat sparse attention significantly reduces computational complexity for long-context processing, while N-gram Embedding enhances token-level representation. The model employs dynamic activation to maintain efficient inference at approximately 48B activated parameters.

A key highlight of this release is that Meituan has simultaneously open-sourced inference code optimized for domestic AI accelerators (Chinese-made GPUs). This substantially lowers the barrier for deploying large-scale code models on China's domestic computing infrastructure.

Meituan also released VitaBench 2.0, the first benchmark for evaluating LLMs on long-term, dynamic user modeling in real-life scenarios — filling a critical gap in agent evaluation methodology.

Industry observers view LongCat-2.0 as a significant step forward in the "agentification" of code models. Its 1.6T parameter scale and long-context optimization provide a new technical pathway for handling large-scale engineering code.
