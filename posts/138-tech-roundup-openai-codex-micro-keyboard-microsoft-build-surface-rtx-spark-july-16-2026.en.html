⚡ July 16 Tech Roundup: OpenAI Launches Codex Micro Keyboard, Microsoft Build 2026 Unveils 'Dream Machine', Tencent Cloud Slashes DeepSeek-V4 Prices Up to 97.5%

1. OpenAI's First Hardware: The Codex Micro Illuminated Keyboard

OpenAI has quietly released its first branded hardware device — not an AR headset or AI smartphone as rumored, but the Codex Micro, an illuminated keyboard designed as a "command center" for AI agents. Each keycap integrates a programmable RGB LED that visually reflects the real-time status of different AI agent threads running on the user's system.

Design: From Keyboard to AI Dashboard

Codex Micro addresses a pain point familiar to modern developers: juggling multiple AI agents for code review, documentation, log monitoring, and other tasks that run in parallel. OpenAI's solution embeds the agent status display into the keyboard itself. Per-key RGB LEDs use different colors and blinking patterns — blue breathing for an active code review agent, red flashing for errors — so users can monitor their agent fleet without looking up from the keyboard.

Technical Specs and Pricing Controversy

According to OpenAI's official blog, the Codex Micro runs proprietary AgentMonitor firmware that communicates with the OpenAI Agent SDK on the host PC. It features an onboard ARM processor and Wi-Fi module for direct cloud agent status reception, a touch bar left of the spacebar for switching agent focus, and a dial on the right for adjusting global workload thresholds.

The pricing has sparked significant controversy — $349 for the base model and $599 for the Pro edition, surpassing many high-end mechanical keyboards. OpenAI product lead Mira Murati defended the price: "Codex Micro isn't a normal keyboard — it's a physical AI status dashboard." However, industry analysts argue the same functionality could be delivered through a software plugin or secondary display.

Notably, the device was co-developed with peripherals brand Keychron (chassis and mechanical switches) rather than built entirely in-house. TechCrunch described it as a "limited-run collaboration" — more of an opening act for OpenAI's hardware ambitions than a flagship product.

2. Microsoft Build 2026: Seven In-House Models and a 'Dream Machine' Dev Box

At Microsoft Build 2026, the company unveiled a comprehensive AI infrastructure lineup spanning models, developer hardware, and new AI terminal form factors.

Seven MAI Models Across Five Domains

Microsoft introduced seven MAI (Microsoft AI) models covering reasoning, code, image, speech, and transcription:

  • MAI Thinking 1: Reasoning model with sparse MoE architecture, 35B active parameters (~1T total), supporting 256K token context
  • MAI Code 1 Flash: End-to-end trained coding model scoring 51.2% on SWE Bench Pro (vs. Claude Haiku 4.5 at 35.2%), rolling out to GitHub Copilot personal users
  • MAI Image 2.5: Image generation model, now in PowerPoint, expanding to OneDrive
  • MAI Transcribe 1.5: Transcription supporting 43 languages
  • MAI Voice 2: Voice model supporting 15 languages with few-shot voice adaptation

Surface RTX Spark Dev Box: Satya Nadella's 'Dream Machine'

On the hardware front, Microsoft unveiled the Surface RTX Spark Dev Box — CEO Satya Nadella's "dream machine." Powered by Nvidia's RTX Spark platform, it delivers 1 petaflop of AI compute, 20 CPU cores, and 128GB of unified memory with a thermal design power of just 100W, launching this fall. Nvidia CEO Jensen Huang appeared via video link, declaring that "the PC is evolving from a personal computer into personal AI."

Microsoft also showcased Project Solara reference devices: a MediaTek-powered desktop terminal with identity recognition and Microsoft 365 Copilot integration, and a Qualcomm wearable chip-based digital badge for mobile healthcare and field service scenarios.

3. OpenAI Expands Codex Ecosystem: Plugins Cover Investment Banking, Data Analysis, and More

At yesterday's "Intelligence at Work" livestream, OpenAI announced it is integrating Codex capabilities directly into the ChatGPT app, alongside role-specific plugins, Sites shared web pages, and annotation features — extending Codex's reach from software development into broader knowledge work.

Official data shows Codex now has over 5 million weekly active users, with non-developers making up roughly 20% and growing 3x faster than developer users.

Six specialized plugins were launched, integrating 62 apps and 110 skills across data analysis, creative production, sales, product design, public equity investment, and investment banking. Future directions include corporate finance, private equity, and legal. The Sites feature (for Enterprise and Business users) lets users generate interactive, shareable web pages from analysis results.

4. Tencent Cloud Slashes DeepSeek-V4 Prices Up to 97.5%

Tencent Cloud announced dramatic price cuts on its DeepSeek-V4 model series, with reductions of up to 97.5% — the latest salvo in China's AI model price war:

  • DeepSeek-V4-Pro: Input and output prices cut 75% to 0.003 yuan and 0.006 yuan per thousand tokens respectively
  • Cache hit price: Slashed 97.5% to just 0.000025 yuan per thousand tokens
  • DeepSeek-V4-Flash: Cache hit price reduced 90% to 0.00002 yuan per thousand tokens

Following similar moves by Alibaba Cloud and Baidu AI Cloud, the price cuts reflect intensifying competition among Chinese cloud providers for inference market share. As open-weight models approach closed-source capability levels, developers are becoming the biggest beneficiaries.

5. Vint Cerf Joins AI Agent Identity Standards Push After Leaving Google

Internet pioneer and "TCP/IP之父" Vint Cerf, after leaving Google where he worked for 20 years, has joined Innovation Labs as an advisor to promote DNSid — an open identity standard for AI agents. The initiative aims to establish a unified identification and audit framework for AI agents operating on the open internet.

Cerf emphasized the critical importance of agent identity: "What authorities do they have? Who is accountable for their behavior?" The DNSid system links each AI agent to an existing internet domain name using cryptographic proofs, with a verifiable registration timeline. Innovation Labs reports it is trialing the standard with several hyperscalers and identity companies.

"I felt like I might be able to help them in a period of time when naming and identification is becoming increasingly important. This is largely triggered by the notion of AI agents and the question of what authorities they have." — Vint Cerf
