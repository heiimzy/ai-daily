⚡ July 23 Tech Double Feature: Alibaba Open-Sources Qwen3-Coder 480B Model Rivalling Claude Sonnet 4, OpenAI Launches ChatGPT Self-Serve Ad Platform

Alibaba Qwen3-Coder: A New Open-Source Benchmark for Agentic Coding

In the early hours of July 23, Alibaba Cloud's Qwen team officially announced and open-sourced its most capable agentic code model to date — Qwen3-Coder-480B-A35B-Instruct. Built on a Mixture-of-Experts (MoE) architecture, the model packs 480 billion total parameters while activating only 35 billion per inference, achieving an optimal balance between performance and computational efficiency.

According to the Qwen team's official benchmarks, Qwen3-Coder achieves state-of-the-art results among open-source models across three critical task categories: Agentic Coding, Agentic Browser-Use, and Agentic Tool-Use. Its comprehensive performance is comparable to Anthropic's flagship closed-source model, Claude Sonnet 4. The model is now available on ModelScope and Hugging Face.

Training: 7.5 Trillion Tokens + 20,000 Parallel Environments

Qwen3-Coder was pre-trained on 7.5 trillion tokens (70% code ratio), excelling in programming while retaining general and mathematical capabilities. It natively supports a 256K token context window, extendable to 1M tokens via YaRN, optimized for repository-scale understanding and dynamic data scenarios such as Pull Request reviews.

For post-training, the team employed a dual-track reinforcement learning strategy. On the Code RL track, they scaled execution-driven large-scale RL across diverse real-world coding tasks, using automated test case generation to create high-quality training instances. On the Agent RL (long-horizon RL) track, they leveraged Alibaba Cloud's infrastructure to build a scalable verification system capable of running 20,000 independent environments in parallel. The result: Qwen3-Coder achieves open-source SOTA on SWE-Bench Verified without any test-time scaling.

Qwen Code CLI Tool Also Released

Alongside the model, Alibaba open-sourced the Qwen Code command-line tool. Forked from Gemini Code, it features custom prompt and function-calling protocol adaptations to fully unleash Qwen3-Coder's capabilities on agentic coding tasks. Developers can install it via npm and use it seamlessly with Claude Code, Cline, and other community tools.

The GitHub repository garnered over 16,000 stars within hours of launch. Alibaba confirmed that more model sizes are on the way, aimed at reducing deployment costs while maintaining strong performance, and that the team is actively exploring self-improving capabilities for the Coding Agent.

OpenAI Opens ChatGPT Ads to All Advertisers

On July 22 (US time), OpenAI officially launched Ads Manager, a self-serve advertising platform for ChatGPT, marking a major strategic shift from subscription-only monetization to ad-supported revenue. Advertisers can now create campaigns, set budgets and goals, and place ads directly within ChatGPT conversations through the self-serve portal.

The move comes roughly two years after OpenAI CEO Sam Altman called advertising a "last resort" business model. Early participating brands include Best Buy, Lowe's, and VistaPrint. Best Buy's VP of Media, Amy Adams, noted that consumers increasingly turn to ChatGPT for research and decision-making, making "being present in these moments crucial."

Conversational Context Targeting with Zero Impact on Answers

Unlike traditional keyword-based search ads, ChatGPT's ad targeting logic is based on conversational context signals. The richer the context users share in conversations, the more precisely ads can be delivered — covering the full intent journey from "exploring options" to "comparing choices" to "making decisions." OpenAI emphasizes that ads run on separate systems and have zero influence on ChatGPT's responses.

The platform supports both CPM (cost-per-mille) and CPC (cost-per-click) bidding models. The campaign workflow involves three steps: create a campaign with budget and goals, upload or create ad details in the tool, then launch and optimize based on performance data.

Free Users See Ads; Plus and Pro Subscribers Are Exempt

OpenAI confirmed that ads are only shown to Free and Go tier users. Plus, Pro, and all Business plan subscribers are fully exempt. Users under 18 are also excluded from ad targeting. Initial rollout markets include the US, UK, Canada, Australia, New Zealand, and Japan.

Industry analysts note that with hundreds of millions of free-tier users, even a ~0.8% ad appearance rate represents massive commercial potential. However, early advertiser feedback suggests results are not yet competitive with established search advertising channels, with some reporting "high costs with limited returns." OpenAI's ad monetization journey remains in its early experimental phase.

Industry Impact

Qwen3-Coder's open-source release demonstrates that Chinese AI labs continue to close the gap with frontier closed-source models. Following Kimi K3's "DeepSeek 2.0 moment," Alibaba's 480B MoE model proves that open-source models can now compete head-to-head with top-tier proprietary systems on cutting-edge agentic coding tasks. For global developers, this means a powerful free alternative to Claude Code.

Meanwhile, OpenAI's ad platform launch signals a critical inflection point — the AI industry is moving from a single "subscription-only" model to a dual-track "ads + subscription" paradigm. Together, these developments underscore that AI competition is shifting from pure capability races to new battlegrounds of business model innovation and ecosystem building.
