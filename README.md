# AI Daily — AI日报

> 📰 每日更新AI领域最新动态，涵盖AI新闻、AI工具推荐、AI赚钱变现方法、AI提升工作效率技巧。

🌐 **Live Site**: https://heiimzy.github.io/ai-daily/

## 📋 内容

- **AI 新闻** — 每日最新AI行业动态、重大事件、政策法规
- **AI 工具推荐** — 精选AI工具评测和使用指南
- **AI 变现** — AI赚钱方法和商业模式分析
- **AI 提效** — AI提升个人和工作效率的技巧

## 🚀 技术

纯静态 HTML/CSS/JS，零服务器成本，通过 GitHub Pages 托管。

### 发布流程

文章通过自动脚本生成并发布：
1. 搜索最新AI新闻动态
2. 撰写中英双语文章（HTML格式）
3. 更新 posts.json 索引
4. 更新 RSS feed
5. 生成 sitemap.xml
6. 推送到 GitHub Pages
